# ESP32-C3 AT Baseline Package

This package turns an ESP32-C3 Super Mini into the standard AT Wi-Fi/MQTT module for the STM32F103 baseline.

Start here:

- `docs/STM32F103_ESP32C3_AT_Baseline_Guide.md`
- `docs/ESP32C3_AT_Flashing_Guide.md`
- `docs/Sensor_Extension_Guide.md`

Firmware:

- `firmware/ESP32-C3-MINI-1-AT-V4.1.1.0.zip`
- `firmware/manifest.json`

Flasher:

- Source: `tools/flasher/esp32c3_at_flasher.py`
- Build EXE: `tools/flasher/build_exe.ps1`
- Release output: `release/ESP32C3_AT_Flasher.exe`
